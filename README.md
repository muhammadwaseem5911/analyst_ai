# AI Data Analyst Assistant

An AI-powered data analysis application that allows users to upload a dataset and ask questions in natural language.  
The system uses an LLM-powered agent to analyze data, generate insights, and create visualizations automatically.

---

## Project Overview

The **AI Data Analyst Assistant** is built with **Streamlit, LangChain, and Groq LLM** to simulate how an AI analyst can explore datasets and generate insights.

Users can upload a dataset and interact with the AI to:

- Analyze data
- Calculate metrics
- Generate charts
- Explore insights using natural language

---

##  Key Features

1. Upload CSV or Excel datasets  
2. Automatic dataset preview  
3. Data cleaning (missing values & duplicates)  
4. Dataset summary statistics  
5. Key metrics calculation (Sales, Profit, Average Sales)  
6. Sales analysis by category  
7. AI-powered data analysis using LangChain agents  
8. Dynamic pandas code generation  
9. Chart generation using Matplotlib  

---

## Tech Stack

- **Python**
- **Streamlit**
- **Pandas**
- **Matplotlib**
- **LangChain**
- **Groq LLM**
---

## How It Works

1️. User uploads a dataset (CSV or Excel)  
2️. The dataset is cleaned and analyzed  
3️. Key metrics are generated automatically  
4️. Users ask questions about the dataset  
5️. The AI agent generates pandas code to analyze the data  
6️. Charts are created using matplotlib and displayed in Streamlit  

---

## Example Capabilities

Example questions the AI can handle:

Total sales in the dataset
Sales by category
Top 5 products by sales
Profit by region
Sales trend analysis

The AI generates **pandas analysis code** and visualizations automatically.
